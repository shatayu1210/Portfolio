## 🚀 Featured Project: AgenticAI AutoBot
**Autonomous GitHub Issue Resolution & Proactive Triage Engine**
An end-to-end multi-agent system that autonomously triages, plans, and patches complex open-source issues.
- **Proactive Alerting:** A Slack-based "Sentinel" that uses a fine-tuned Scorer/Reasoner to detect high-risk issues in real-time, providing cited reasoning for early reprioritization.
- **Autonomous Patching:** Multi-agent LangGraph workflow (Planner/Patcher/Critic) that generates and validates code fixes.
- **Key Tech:** LangGraph, Neo4j (GraphRAG), Fine-tuned Qwen2.5 (LoRA), Go (Indexing), Prometheus/Grafana.
