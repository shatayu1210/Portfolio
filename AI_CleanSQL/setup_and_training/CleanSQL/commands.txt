# Local Steps

1) Install required dependencies
`pip install -r requirements.txt`

2) Start vLLM (new terminal)
`bash scripts/serve_vllm.sh`

3) Prepare Spider 1.0 (placing SQLite DBs & dev JSON under `data/spider_databases` and `data/dev_gold.json`)
Download from https://drive.google.com/file/d/1403EGqzIDoHMdQF4c9Bkyl7dZLZ5Wt6J/view
unzip spider_data.zip
create data folder inside CleanSQL
move spider_data/database to data as spider_databases  
move spider_data/dev.json to data as dev_gold.json

4) Select Stage-1 set (Clean spider dbs with a comprehensive variety of nl-sql pairs)
`bash scripts/select_stage1.sh`

5) Build RAG index (profiling the Stage-1 DBs, then building RAG index)
`bash scripts/build_rag.sh`

6) Run **prompt_engineering_eval** (Stage-1, RAG ON, topK=3)
`bash scripts/run_all_stage1.sh`

7) Select Stage-2 DBs & queries; **synthesize corrupted set** (reproducible)
`bash scripts/select_stage2.sh`

8) Run **rag_norag_eval** (full flow; CoT only) on Stage-2 DBs to evaluate closeness for corrupted data vs clean
`python -m cleansql.eval.run_stage2 --context schema_only --out results/stage2_schema_only.csv`

9) Run **rag_rag_eval** (full flow; CoT only)
`python -m cleansql.eval.run_stage2 --context rag_topk3 --out results/stage2_rag_k3.csv`
`python -m cleansql.eval.run_stage2 --context rag_topk5 --out results/stage2_rag_k5.csv`
`python -m cleansql.eval.run_stage2 --context rag_topk8_rerank --out results/stage2_rag_k8r.csv`

10) Aggregate all results
`python -m cleansql.eval.aggregate --stage1_glob "results/stage1_*.csv" --stage2_glob "results/stage2_*.csv"`

11) Optional **lora_peft_finetune** (A100 recommended; L4 works with QLoRA)
`bash scripts/peft_train.sh`
